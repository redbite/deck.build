<section>
<b>deck.build</b> is a web project designed to build custom decks by searching your favourite TCG cards.<br>
<br>
<h2>INDEX</h2>
<p align="center">
<img src="https://i.imgur.com/IdW7wKN.png" style="width:1000px;">
</p>
The landing page lets the user access the home page by sign-in or login.
<h2>HOME</h2>
<p align="center">
<img src="https://i.imgur.com/EbFTnDg.png" style="width:1000px;">
</p>
From the home the user has 5 different options:
<ul>
    <li>Create deck</li>
    <li>Delete deck</li>
    <li>View deck</li>
    <li>Search cards</li>
    <li>Logout</li>
</ul>
<h2>SEARCH CARDS</h2>
<p align="center">
<img src="https://i.imgur.com/YEQIUpH.png" style="width:1000px;">
</p>
From the search it is possible to add a card, up to maximum 20 in quantity, to an existing deck.
<h2>VIEW DECK</h2>
<p align="center">
<img src="https://i.imgur.com/6MAPubJ.png" style="width:1000px;">
</p>
It's possible to remove a card via the Remove button.
The suggested amount of cards for deck is in the range 40,60. 
The card with the highest HP is selected to be the leader of the deck.<br>
</section>
